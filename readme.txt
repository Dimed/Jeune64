Manuel d'utilisation :

    -Utiliser une version recente de linux
    -Utiliser une version recente de chrome

    -possédé une version recente de php ( si non possédé l'installer avec la commande : sudo apt install php )
    -possédé une version recente de ckhtmltopdf ( si non possédé l'installer avec l'aide de ce tuto : https://computingforgeeks.com/install-wkhtmltopdf-on-ubuntu-debian-linux/ )

    -installer et decompresser l'archive du jeu
    -se placer dans le repertoire Jeune64 et executé : php -S localhost:8000

    -Ouvrire une page sur chrome d'url : http://localhost:8000
